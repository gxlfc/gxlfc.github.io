    # 一段代码开始

    function _extends() {
    _extends = Object.assign || function (target) {
        for (var i = 1; i < arguments.length; i++) {
            var source = arguments[i];
            for (var key in source) {
                if (Object.prototype.hasOwnProperty.call(source, key)) {
                    target[key] = source[key];
                }
            }
        }
        return target;
    };
    return _extends.apply(this, arguments);
}

    上面的函数内部定义了一个_extends变量，然后使用apply调用_extends(所以很显然这是一个函数)，同时传入了执行上下文this和参数arguments。

    那么重点在于这个_extends变量是什么？我们抽丝剥茧来看，最外面的层级是一个Object.assign || 匿名函数。||逻辑，其实就是看第一个值的真假，真的话，就不会再判断后面的值，假的话才会执行后面的逻辑。那么其含义就显而易见了，首先检测Object.assign方法是否存在，如果存在就使用该函数，如果不存在则我们自己定义一个新函数。

    Object.assign ： 我们都知道其作用是（参看MDN）将所有可枚举属性的值从一个或多个源对象复制到目标对象，并且返回目标对象。很显然我们可以大概猜测出来后面函数在做什么了，其作用必是拷贝对象属性；

    // 对入参数组（对象数组）做一个for循环;
var source = arguments[i]; //source = 单个参数对象
for (var key in source) { // 对source做key-value循环
    // Object.prototype.hasOwnProperty.call(source, key)
    // 其含义是检测被赋值对象是否包含该属性（自身属性，非原型链属性）。
    // 如果存在，才对相应key做赋值操作，很显然赋值操作会覆盖原有value.
    if (Object.prototype.hasOwnProperty.call(source, key)) {
        target[key] = source[key];
    }
}

    注：由于只是循环了1次，并未对value值进行再次循环，那么其便利深度为1；

    综上所述，我们就可以大致得到结论了，其函数的作用为，依次查看入参对象的key，如果被赋值的对象存在该key，则覆盖，后面的永远会覆盖前面；
