状态机 & 遍历器对象生成函数，执行generator函数，返回的是一个iterator对象，指向内部状态的指针对象；

  • 特性： *yield
  • next（）运行返回value逻辑：
      1. 遇到yield，暂停执行后面操作，并将紧跟yield后的表达式作为返回value值；
      1. 下次调用next()才继续执行下去，直到遇到下一个yield；
      1. 没有遇到新的yield语句，一直到函数结束，直到return为止，返回return表达式的值；
      1. 没有return 返回undefined；
      1. 如果先遇到了rerun，返回return值后，不知执行后面的语句了；







 









function* returnBefore() {
  	yield 1;
  	return 111;
  	yield 2;
}
var b = returnBefore();
console.log(b.next()); { value: 1, done: false }
console.log(b.next()); { value: 111, done: true }
console.log(b.next()); { value: undefined, done: true }
console.log(b.next()); { value: undefined, done: true }

# next方法，可携带参数

  • yield语句本身没有返回值，或者说总是返回undefined。var a = yield '11'; // a=undefined
  • next 方法可以带有一个参数，该参数会当做上一条yield语句的返回值。 所以不能在第一次调用next()时传参进去；
function* foo(x) {
    var y = 2 * (yield (x + 1));
    var z = yield(y / 3);
    return x + y + z;
}
var a = foo(5);
a.next(); // 6
a.next(12); // 把上一条yield (x+ 1) 置为12，y=24, 则yield返回24/4
a.next(13); // 把上一条z置为了13，5 + 24 + 13 = 42

# Async & Await

Generator函数的语法糖

async function getData () {
    let res = await requestData();
}

await 后面跟promise、数字、字符串、布尔值，后三者会经Promise.resolve转化后同步执行。

  • 整体逻辑：遇到await先返回，等到异步操作完成，再执行函数体后面的语句；
  • async返回是一个promise对象，如果函数内部return/内部出错，其promise状态会提前onfulfilled，否则会等所有异步任务完成；
  • 函数内部的return语句的返回值，会成为后续then函数的参数；

其中一个await语句后面的promise变成reject，整个async都会中断执行。

aysnc function() { 
    await Promise.reject('出错了');
    await task1(); // 不会被执行
}

如何可以不中断后面的异步任务？

// try catch包装
aysnc function() {
    try {
        await Promise.reject('出错了'); 
    } catch(err) {}
    await task1(); // 会被执行
}

// 错误捕获
aysnc function() {
    await Promise.reject('出错了').catch(err => {}); 
    await task1(); // 会被执行
}

# 需求1：分块上传

const PIECE_SIZE = 1 * 1024 * 1024; // 单位MB
let file = new File(["foo"], "foo.txt");
sliceFileAndUpload();

async sliceFileAndUpload() {
    let dataArr = [];
    for (let start = 0; start < file.size; start += PIECE_SIZE) {
        // 文件切片
        const chunk = file.slice(start, start + PIECE_SIZE);
        try {
            let formData = new FormData();
            formData.append('partFile', chunk);
            // 开始上传
            let res = await axios.post('/upload', formData);
            if (+res.errno !== 0 || !res.data) {
                return Promise.reject('upload failed~~~');
            }
            dataArr.push(res.data);
        } catch (err) {
            return Promise.reject('upload catch error~~~');
        }
    }
    return dataArr;
}
