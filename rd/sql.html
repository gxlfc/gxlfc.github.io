# 查找

  • =、 !=/ <>、>、 < 、>= 、<=、多个条件之间可以用or、and、not等逻辑运算符
  • %：表示任意 0 个或多个字符，_：表示任意单个字符
    • %a 以a结尾的数据；a% 以a开头的数据；%a% 含有a的数据
    • _a_ 三位且中间字母是a的；_a 两位且结尾字母是a的；a_ 两位且开头字母是a的
  • []：表示括号内所列字符中的一个
  • [^] ：表示不在括号所列之内的单个字符
  • BINARY 关键字来设定 WHERE 子句的字符串比较是区分大小写的 where binary a = 'TEST'
  • in (const1, ..., constN) 、 not in (const1, ..., constN)
  • between num1 and num2

结果限制：

  • order by
  • limit pageSize offset pageIndex*pageSize
函数名 COUNT SUM AVG MAX MIN CONCAT

# COLUMN

  • CHANGE

    • 可以重命名列并更改其定义，或同时更改两者。具有比MODIFY或RENAME COLUMN更大的功能，但是牺牲了某些操作的便利性。 CHANGE如果不重命名，则需要两次命名该列，如果仅重命名，则需要重新指定列定义。
    • 使用FIRST或AFTER，可以对列进行重新排序。

  • MODIFY：

    • 可以更改列定义，但不能更改其名称。
    • 比change更方便的更改列定义而不重命名。
    • 使用FIRST或AFTER，可以对列进行重新排序。

  • RENAME

    • 可以更改列名，但不能更改其定义。
    • 重命名列而不更改其定义，比CHANGE更方便。

  • ALTER

    • 仅用于更改列默认值。
